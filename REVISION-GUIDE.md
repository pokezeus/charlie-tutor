# 📚 Charlie Tutor - HK Curriculum Revision Guide

## Primary 3 (P3) Mathematics

### Multiplication Tables (1-10)
**HK Primary 3 Curriculum Standard**

#### Key Concepts:
- Understanding multiplication as repeated addition
- Mastering times tables 1-10
- Applying multiplication in real-world problems

#### Learning Progression:
1. **Tables 1-5** (Term 1)
   - 1×: Identity property
   - 2×: Doubling
   - 5×: Counting by 5s
   - 10×: Place value pattern

2. **Tables 6-10** (Term 2)
   - 6×, 7×, 8×: Building on known facts
   - 9×: Digit sum pattern (9, 18, 27, 36...)
   - Mixed practice

#### Common Patterns & Tricks:
- **9× trick**: Digits add to 9 (9×7=63, 6+3=9)
- **5× pattern**: Ends in 0 or 5
- **10× pattern**: Add a zero
- **Commutative property**: 6×7 = 7×6

## Kindergarten 3 (K3) Mathematics

### Addition & Subtraction (within 20)
**HK K3 Curriculum Standard**

#### Key Concepts:
- Counting objects 1-20
- Simple addition (combining groups)
- Simple subtraction (taking away)
- Number recognition and formation

#### Learning Progression:
1. **Numbers 1-10**
   - Counting objects
   - Number recognition
   - Simple addition within 10

2. **Numbers 11-20**
   - Place value introduction (tens and ones)
   - Addition within 20
   - Subtraction within 20

#### Visual Aids:
- Finger counting
- Number lines
- Counting blocks
- Picture representations

## Revision Tips for Parents

### Daily Practice (15-20 mins):
1. **Morning**: Quick flashcard review (5 min)
2. **After school**: Game-based practice (10 min)
3. **Evening**: Review mistakes together (5 min)

### Weekly Goals:
- **P3**: Master 2 multiplication tables per week
- **K3**: Learn 5 new addition facts per week
- **Both**: Maintain streak for consistency

### Progress Tracking:
- Monitor accuracy % in games
- Track streak days
- Review parent dashboard insights
- Celebrate achievements
